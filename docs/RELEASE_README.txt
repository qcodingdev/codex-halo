Codex Halo v0.1.8
=================

Your screen glows with Codex.

macOS
-----
1. Extract the entire ZIP.
2. Run "Install Codex Halo.command".
3. First launch: right-click "Codex Halo.app", then choose Open.
4. Use Codex normally; Halo detects the local task lifecycle automatically.
5. From the menu-bar icon, choose Demo Mode.

Windows
-------
1. Extract the entire ZIP.
2. Run "Install-Codex-Halo.ps1" with PowerShell.
3. Use Codex normally; no Hook or trust configuration is required.
4. From the tray icon, choose Demo Mode.

Privacy and safety
------------------
Codex Halo is local-only. It has no network service and no telemetry. It reads
only local task lifecycle record types, never prompts, source code, tool input,
or responses. Installation removes only obsolete Halo hook entries if present.

This is an independent community project and is not affiliated with or endorsed
by OpenAI.

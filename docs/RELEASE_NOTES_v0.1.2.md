# Codex Halo 0.1.2

## Polished first-run guidance

The bundled macOS and Windows installers no longer send people to `/hooks`.
They install the correct current Codex Desktop configuration automatically. If
Codex needs executable-hook approval, it presents its own native one-time
confirmation on first use; users never need to locate or edit a Hook file.

This is a documentation and installer-flow polish release on top of v0.1.1's
activation, multi-display, branded breathing-light, and README visual updates.
